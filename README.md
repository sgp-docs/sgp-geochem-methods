# SGP Geochemical Methods

This is the home of a **work in progress** wiki 

https://github.com/ufarrell/sgp-geochem-methods/wiki

used to keep notes about geochemical methods as new data is added to SGP Phase 3. It is based on the [summaries made for SGP Phase 2](https://github.com/ufarrell/sgp_phase2/wiki/C.-Analyses), which will remain static and continue to be the reference for the data made available in the [Phase 2 data release](http://sgp-search.io/).

Details about SGP (Sedimentary Geochemistry and Paleonenvironments Project) can be found here: https://sgp.stanford.edu/

SGP Phase 2 data was released in Feb 2025 and can be searched here: http://sgp-search.io/

Details of the Phase 2 dataset are described here: https://github.com/ufarrell/sgp_phase2
