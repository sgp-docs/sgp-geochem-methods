# sgp_analytical_methods

This is a **work in progress**. It is used to keep notes about geochemical methods as new data is added to the SGP Phase 3 dataset. It is based on the [summaries made for SGP Phase 2](https://github.com/ufarrell/sgp_phase2/wiki/C.-Analyses), which will remain static and continue to be the reference for the data made available in the [Phase 2 data release](http://sgp-search.io/).
